# E-Health Management Website

This project is a front-end implementation of an **E-Health Management System** designed to demonstrate how healthcare services can be managed through a digital platform.

## Features
- Secure Login Interface
- Patient Dashboard
- Online Appointment Booking
- Contact Us Page for Queries
- Clean and User-Friendly Design
- Organized using External CSS for Styling
          
## Technologies Used
- HTML5
- CSS3
- Visual Studio Code

## Project Objective
The objective of this project is to digitalize basic healthcare interactions and provide an accessible interface where patients can manage appointments and view information efficiently.

## Domain
E-Health (Healthcare Information Systems)
